# Инструкция: Многоагентная Аналитическая Система ( MAS Sber)

Добро пожаловать в многоагентную систему, спроектированную для автоматического контент-анализа, проверки информационной безопасности и агрегации результатов. Система построена на базе паттерна **Orchestrator-Workers (Оркестратор и Исполнители)** с единой точкой координации и встроенным почтовым шлюзом для отправки сводных отчетов.

## 1. Архитектура Системы

Система состоит из Единого Координатора (Orchestrator) и 4-х специализированных субагентов-ролей:

```
                  ┌──────────────────────────────┐
                  │   Координатор-Оркестратор    │
                  └──────────────┬───────────────┘
                                 │
         ┌───────────────────────┼───────────────────────┐
         ▼                       ▼                       ▼
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│ Субагент 1:     │     │ Субагент 2:     │     │ Субагент 3:     │
│ Анализ дублей   │     │ Проверка ПДн    │     │Анализ отзывов   │
└────────┬────────┘     └────────┬────────┘     └────────┬────────┘
         │                       │                       │
         └───────────────────────┼────────────────────────┘
                                 ▼
                     ┌───────────────────────┐
                     │ Субагент 4: Почтальон │
                     └───────────────────────┘
```

1. **Роль Координатора (Orchestrator):** Принимает запросы, управляет жизненным циклом и последовательным запуском задач, передает временные артефакты между субагентами и формирует финальный сводный дайджест.
2. **Анализатор дублей (Duplicates Analyzer):** Оценивает степень сходства заголовков и описаний учебных программ из Excel, проверяет маркеры целевой аудитории, уровни сложности и кавычки. Ставит статусы проверки.
3. **Анализатор утечек ПДн (PII Analyzer):** Анализирует лендинги и маркетинговые тексты на предмет случайного попадания конфиденциальних персональных данных (ФИО, номера телефонов, банковские карты) с верификацией по алгоритму Луна.
4. **Анализатор комментариев (Comments Analyzer):** Извлекает из фидбека баги и ключевые проблемы (боли) контента, выставляет тональность, подсчитывает коэффициент проблемности и формирует емкие рекомендации.
5. **Агент-почтальон (Postman):** Упаковывает текстовые сводки и файлы отчетов в красивое HTML/Plain-text сообщение с вложениями и отправляет на целевой адрес.

## 2. Требования к Входным Данным и Файлам

Агенты настроены на автоматический рекурсивный поиск файлов в рабочей директории, в которой запущен скрипт, или на Рабочем столе. 

### Файл 1: Для Анализа Дублей
* **Название файла:** `Свод электронного контента.xlsx`
* **Обязательные колонки (Названия колонок регистронезависимы):**
  * `ID курса` — уникальный строковый или числовой идентификатор.
  * `Название курса` — текст заголовка для очистки.
  * `Описание курса` — текст описания для извлечения ключевых слов.

### Файл 2: Для Анализа Персональных Данных (ПДн)
* **Название файла:** `landing_text` (допускаются форматы `.docx`, `.doc`, `.txt`).
* **Что ищет агент:**
  * **Телефоны:** Любые последовательности от 10 до 16 цифр (игнорируя дефисы, скобки, пробелы и плюсы). Учитывает маски маскирования с буквой "Х" (например, `9ХХ`).
  * **ФИО:** Цепочки из 2-3 слов на основе словаря фамилий, имен и патронимов (окончания `-ич`, `-вна`, `-овна`). Игнорирует тестовые строки (если перед ними стоят слова `тестовый`, `образцовый` и т.д.).
  * **Карты:** 16-значные последовательности, проходящие проверку по формуле Луна ( Luhn Algorithm ) и начинающиеся на 4 (Visa), 5 (Mastercard) или 2 (Мир).

### Файл 3: Для Анализа Комментариев и Обратной Связи
* **Название файла:** `Комментарии по курсам_синтетика.xlsx`
* **Обязательные колонки:**
  * `ID` или `Номер` — уникальный ID отзыва.
  * `Комментарий` — текст отзыва.
* **Что делает агент:**
  * Очищает текст от эмодзи и спецсимволов.
  * Классифицирует баги по триггерам (`не грузит`, `вылетает`, `ошибка`, `завис`, `не открывается`, `белый экран`).
  * Определяет боли контента (`сложно`, `непонятно`, `мало практики`, `много воды`).
  * Считает коэффициент проблемности и выделит Top-3 багов и Top-3 болей.

## 3. Описание логики работы и Алгоритмов

### Очистка и Сверка Дублей
Агент 1 приводит тексты курсов к нижнему регистру, избавляется от слов-паразитов («курс», «онлайн-программа», «вебинар») и удаляет точки на концах.
Далее сверяет курсы по 3 уровням приоритета:
1. **Точный дубль** (очищенные названия абсолютно идентичны).
2. **Вложенное Название** (одно очищенное название полностью входит в другое).
3. **Вложенное Описание** (описания содержат как минимум 2 общих смысловых слова длиной от 4 символов).

**Стоп-фильтры (переводят в статус «Требует внимания»):**
* Если тексты внутри кавычек в названиях различаются.
* Если один курс содержит маркер аудитории (например, *«для подростков»*), а второй — нет, либо маркеры аудитории различаются.
* Если маркеры уровня сложности противоположны (*«Базовый»* vs *«Продвинутый»*).

## 4. Как Запустить Систему

Для запуска убедитесь, что у вас установлена библиотека для чтения и записи Excel. Если библиотеки отсутствуют, система автоматически упакует отчеты в кодировке UTF-8 с разделителем `;` в формате `.csv` для совместимости.

### Установка зависимостей (желательно, но не обязательно)
```bash
pip install pandas openpyxl python-docx
```

### Настройка SMTP для отправки на реальную почту
Откройте файл `multiagent_system.py` и в конструкторе `PostmanAgent` или при запуске укажите ваши SMTP-параметры. Если параметры не переданы, скрипт сгенерирует красивый полностью готовый файл письма в формате `.eml` прямо на вашем компьютере. Вы сможете открыть его в любом почтовом клиенте (Outlook, Почта macOS/Windows).

### Запуск
Перейдите в папку с файлом и запустите терминал Python:
```bash
python3 multiagent_system.py
```

### Результаты Работы:
На выходе система создаст:
1. `Отчет_по_дублям_каталога_ГГГГ-ММ-ДД.xlsx` — структурированный список дублирующихся курсов с разбивкой стоп-фильтров.
2. `Отчет_по_ПДн_документы_ГГГГ-ММ-ДД.xlsx` — реестр утекших телефонов, ФИО и карт с рекомендациями.
3. `Mail_Digest_ГГГГ-ММ-ДД.eml` — собранный красивый почтовый дайджест с вложенными отчетами Excel.
